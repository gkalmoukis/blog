### Εισαγωγή δεδομένων
Η εισαγωγή δεδομένων από το πληκτρολόγιο γίνεται με την εντολή: `ΔΙΑΒΑΣΕ` Η εντολή `ΔΙΑΒΑΣΕ` ακολουθείται πάντοτε από ένα ή περισσότερα ονόματα μεταβλητών. Αν υπάρχουν περισσότερες από μία μεταβλητές τότε αυτές χωρίζονται με κόμμα `,`.
```:title=Σύνταξη
ΔΙΑΒΑΣΕ <λίστα_μεταβλητών> 
```

```:title=Παράδειγμα
ΔΙΑΒΑΣΕ Ηλικία
ΔΙΑΒΑΣΕ χ, ψ, ω
```

### Εμφάνιση των δεδομένων
Η εμφάνιση των δεδομένων στην οθόνη γίνεται με την εντολή: `ΓΡΑΨΕ`. Η εντολή `ΓΡΑΨΕ` ακολουθείται πάντοτε από ένα ή περισσότερα ονόματα στοιχείων. Μπορούν να υπάρχουν σταθερές τιμές (μέσα σε μονά εισαγωγικά `''`) και ονόματα μεταβλητών. Τα στοιχεία χωρίζονται με κόμμα `,`.

Σύνταξη Παράδειγμα
```:title=Σύνταξη
ΓΡΑΨΕ <λίστα_στοιχείων> 
```

```:title=Παράδειγμα
ΓΡΑΨΕ Ηλικία
ΓΡΑΨΕ 'Το χ είναι: ', χ
```

### Δομή προγράμματος

Κάθε πρόγραμμα σε ΓΛΩΣΣΑ πρέπει να έχει κάποια ή όλα από τα παρακάτω τμήματα:
- **Επικεφαλίδα** του προγράμματος (απαραίτητο)
- Τμήμα δήλωσης **σταθερών**
- Τμήμα δήλωσης **μεταβλητών**
- **Αρχή** κύριου μέρους προγράμματος (απαραίτητο)
- Λίστα **εκτελέσιμων εντολών** (απαραίτητο)
- **Τέλος** κύριου μέρους προγράμματος (απαραίτητο)


```:title=Σύνταξη
ΠΡΟΓΡΑΜΜΑ <όνομα_προγράμματος>
ΣΤΑΘΕΡΕΣ
    <όνομα_σταθεράς> = <τιμή>
    ...
ΜΕΤΑΒΛΗΤΕΣ
    <τύπος_δεδομένων>: Μετ1, Μετ2
    ...
ΑΡΧΗ
    <λίστα_εντολών>
ΤΕΛΟΣ_ΠΡΟΓΡΑΜΜΑΤΟΣ <όνομα_προγράμματος>
```

Για Παράδειγμα για να γράψουμε ενα πρόγραμμα σε ΓΛΩΣΣΑ που να ζητάει ένα ποσό απο τον χρήστη και να υπολογίζει την τιμή με Φ.Π.Α θα πρέπει να γράψουμε τον παρακάτω κώδικα.

```:title=ΓΛΩΣΣΑ
ΠΡΟΓΡΑΜΜΑ ΦΠΑ
ΣΤΑΘΕΡΕΣ
    ΦΠΑ = 0.23
ΜΕΤΑΒΛΗΤΕΣ
    ΠΡΑΓΜΑΤΙΚΕΣ : ποσο, κοστος
ΑΡΧΗ
    ΔΙΑΒΑΣΕ ποσο
    κοστος <- ποσο * (1 + ΦΠΑ)
    ΓΡΑΨΕ 'Το συνολικό κόστος είναι: ', κοστος
ΤΕΛΟΣ_ΠΡΟΓΡΑΜΜΑΤΟΣ ΦΠΑ
```
### Δομή ακολουθίας στη ΓΛΩΣΣΑ

Στο δομημένο προγραμματισμό χρησιμοποιούμε μόνο τρεις λογικές δομές. Η απλούστερη είναι η δομή ακολουθίας. Χρησιμοποιείται για να λύσουμε απλά προβλήματα, όταν έχουμε δεδομένη τη σειρά εκτέλεσης των ενεργειών. Οι εντολές εκτελούνται σειριακά η μία μετά την άλλη από πάνω προς τα κάτω. 

Μπορούμε να έχουμε τις εντολές:
- Εισόδου
- Εκχώρησης τιμής
- Εξόδου

```:title=ΓΛΩΣΣΑ
ΠΡΟΓΡΑΜΜΑ Μέσος_όρος
ΜΕΤΑΒΛΗΤΕΣ
    ΑΚΕΡΑΙΕΣ: α, β, γ
ΠΡΑΓΜΑΤΙΚΕΣ: μο
ΑΡΧΗ
    ΔΙΑΒΑΣΕ α, β, γ
    μο <- (α + β + γ) / 3
    ΓΡΑΨΕ 'Ο μέσος όρος είναι: ', μο
ΤΕΛΟΣ_ΠΡΟΓΡΑΜΜΑΤΟΣ Μέσος_όρος
```
