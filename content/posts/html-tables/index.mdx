### Εισαγωγή στους πίνακες

Υπάρχουν πολλοί ιστότοποι στο διαδίκτυο που εμφανίζουν πληροφορίες όπως τιμές μετοχών, αθλητικές βαθμολογίες, δεδομένα τιμολογίων και άλλα. 
Αυτά τα δεδομένα έχουν φυσική μορφή πίνακα, πράγμα που σημαίνει ότι ένας πίνακας είναι συχνά ο καλύτερος τρόπος παρουσίασης των δεδομένων.

Θα δούμε πώς να χρησιμοποιούμε το element της HTML `<table>` για την παρουσίαση πληροφοριών σε έναν δισδιάστατο πίνακα στους χρήστες.
 
Δημιουργία ενός πίνακα
 
Πριν από την εμφάνιση δεδομένων, πρέπει πρώτα να δημιουργήσουμε τον πίνακα που θα περιέχει 
τα δεδομένα χρησιμοποιώντας το element `<table>`.

```html noLineNumbers
<table>
</table>
```

Το elememt `<table>` θα περιέχει όλα τα δεδομένα πίνακα που σκοπεύουμε να εμφανίσουμε.

### Σειρές πίνακα

Σε πολλά προγράμματα που χρησιμοποιούν πίνακες, ο πίνακας είναι ήδη προκαθορισμένος για εσάς, 
πράγμα που σημαίνει ότι περιέχει τις σειρές, τις στήλες και τα κελιά που θα διατηρούν δεδομένα. 
Στην HTML, όλα αυτά τα στοιχεία πρέπει να δημιουργηθούν.

Το πρώτο βήμα για την εισαγωγή δεδομένων στον πίνακα είναι να προσθέσετε σειρές χρησιμοποιώντας το elememt νέας γραμμής πίνακα: `<tr>`.

```html noLineNumbers {2,3}
<table>
  <tr>
  </tr>
</table>
```
Στο παραπάνω παράδειγμα, στον πίνακα έχει προστεθεί μία σειρά.

### Δεδομένα πίνακα

Οι σειρές δεν επαρκούν για την προσθήκη δεδομένων σε έναν πίνακα. 
Κάθε στοιχείο κελιού πρέπει επίσης να οριστεί. 
Στην HTML, μπορείτε να προσθέσετε δεδομένα χρησιμοποιώντας το elememt δεδομένων πίνακα: `<td>`.

```html noLineNumbers {3,4}
<table>
  <tr>
    <td> a </td>
    <td> b </td>
  </tr>
</table>
```

Στο παραπάνω παράδειγμα, δύο σημεία δεδομένων (**a** και **b**) καταχωρήθηκαν στη μία σειρά που υπάρχει. 
Προσθέτοντας δύο σημεία δεδομένων, δημιουργήσαμε δύο κελιά δεδομένων.

Εάν ο πίνακας εμφανιζόταν στο πρόγραμμα περιήγησης, θα εμφανιζόταν ένας πίνακας με μία σειρά και δύο στήλες.

### Επικεφαλίδες πίνακα

Τα δεδομένα πίνακα δεν έχουν νόημα χωρίς τίτλους για να περιγράψουν τι αντιπροσωπεύουν τα δεδομένα.

Για να προσθέσετε τίτλους σε γραμμές και στήλες, μπορείτε να χρησιμοποιήσετε το στοιχείο επικεφαλίδας πίνακα: `<th>`.

Το στοιχείο κεφαλίδας πίνακα χρησιμοποιείται ακριβώς όπως ένα στοιχείο δεδομένων πίνακα, εκτός από έναν σχετικό τίτλο. 
Ακριβώς όπως τα δεδομένα πίνακα, μια επικεφαλίδα πίνακα πρέπει να τοποθετηθεί μέσα σε μια σειρά πίνακα.

```html {3,4,5,8}
<table>
  <tr>
    <th></th>
    <th scope="col">Header 1</th>
    <th scope="col">Header 2</th>
  </tr>
  <tr>
    <th scope="row">Row header</th>
    <td>a</td>
    <td>b</td>
  </tr>
</table>
```

Τι συνέβη στον παραπάνω κώδικα;

1. Πρώτον, προστέθηκε μια νέα σειρά για να κρατήσει τις τρεις επικεφαλίδες: μια κενή επικεφαλίδα, μια επικεφαλίδα *Header 1* και μια επικεφαλίδα της *Header 2*. 
Η κενή επικεφαλίδα δημιουργεί το επιπλέον κελί πίνακα που απαιτείται για να ευθυγραμμιστούν σωστά οι επικεφαλίδες του πίνακα με τα δεδομένα στα οποία αντιστοιχούν.

2. Στη δεύτερη σειρά, ένας τίτλος πίνακα προστέθηκε ως τίτλος γραμμής: *Row header*.

> Σημειώστε επίσης, τη χρήση του χαρακτηριστικού εύρους, το οποίο μπορεί να λάβει μία από τις δύο τιμές:

> `row` αυτή η τιμή καθιστά σαφές ότι η επικεφαλίδα είναι για μια σειρά.
> `col` αυτή η τιμή καθιστά σαφές ότι η επικεφαλίδα είναι για μια στήλη.

> Ο κώδικας HTML για πίνακες μπορεί να φαίνεται λίγο περίεργος στην αρχή, αλλά η ανάλυσή του ανά κομμάτι βοηθά τον κώδικα να γίνει πιο κατανοητό.

### Εκτεταμένες στήλες

Τι γίνεται αν ο πίνακας περιέχει δεδομένα που εκτείνονται σε πολλές στήλες;

Για παράδειγμα, ένα προσωπικό ημερολόγιο μπορεί να έχει συμβάντα που εκτείνονται σε πολλές ώρες ή ακόμη και σε πολλές ημέρες.

Τα δεδομένα μπορούν να εκτείνονται σε στήλες χρησιμοποιώντας το χαρακτηριστικό colspan. 
Τα attributes δέχονται έναν ακέραιο (μεγαλύτερο ή ίσο με 1) για να δηλώσουν τον αριθμό των στηλών που εκτείνεται.

```html {8}
<table>
  <tr>
    <th>Monday</th>
    <th>Tuesday</th>
    <th>Wednesday</th>
  </tr>
  <tr>
    <td colspan="2">Out of Town</td>
    <td>Back in Town</td>
  </tr>
</table>
```

Στο παραπάνω παράδειγμα, το κελί *Out of Town* καλύπτει τις επικεφαλίδες του πίνακα *Monday* και *Monday* χρησιμοποιώντας την τιμή 2 (δύο στήλες). 
Το *Back in Town* εμφανίζεται μόνο στην επικεφαλίδα της Τετάρτης.

### Εκτεταμένες σειρές

Τα δεδομένα μπορούν επίσης να εκτείνονται σε πολλές σειρές χρησιμοποιώντας το χαρακτηριστικό `rowspan`.

Το χαρακτηριστικό `rowspan` χρησιμοποιείται για δεδομένα που εκτείνονται σε πολλές σειρές (ίσως ένα συμβάν συνεχίζεται για πολλές ώρες σε μια συγκεκριμένη ημέρα). 
Δέχεται έναν ακέραιο (μεγαλύτερο ή ίσο με 1) για να δηλώσει τον αριθμό των σειρών που εκτείνεται.

```html 
<table>
  <tr>
    <th></th>
    <th>Saturday</th>
    <th>Sunday</th>
  </tr>
  <tr>
    <th>Morning</th>
    <td rowspan="2">Work</td>
    <td rowspan="3">Relax</td>
  </tr>
  <tr>
    <th>Afternoon</th>
  </tr>
  <tr>
    <th>Evening</th>
    <td>Dinner</td>
  </tr>
</table>
```

Στο παραπάνω παράδειγμα, υπάρχουν τέσσερις σειρές:

1. Η πρώτη σειρά περιέχει ένα κενό κελί και τις επικεφαλίδες των δύο στηλών.
2. Η δεύτερη σειρά περιέχει την επικεφαλίδα της σειράς *Morning*, μαζί με το *Work*, το οποίο εκτείνεται σε δύο σειρές κάτω από τη στήλη του *Saturday*. Η καταχώριση *Relax* εκτείνεται σε τρεις σειρές κάτω από τη στήλη της *Sunday*.
3. Η τρίτη σειρά περιέχει μόνο την επικεφαλίδα της γραμμής *Afternoon*.
4. Η τέταρτη σειρά περιέχει μόνο την είσοδο του *Dinner*, καθώς το *Relax* εκτείνεται στο κελί δίπλα του.


### Σώμα πίνακα

Με την πάροδο του χρόνου, ένας πίνακας μπορεί να αναπτυχθεί για να περιέχει πολλά δεδομένα και να γίνει πολύ μεγάλος. 
Όταν συμβεί αυτό, ο πίνακας μπορεί να διαχωριστεί έτσι ώστε να είναι ευκολότερη η διαχείριση.

Oι πίνακες μπορούν να διαχωριστούν χρησιμοποιώντας το στοιχείο σώματος του πίνακα: `<tbody>`.

Το στοιχείο `<tbody>` πρέπει να περιέχει όλα τα δεδομένα του πίνακα, 
εξαιρουμένων των επικεφαλίδων του πίνακα.

```html {2,20}
<table>
  <tbody>
    <tr>
      <th></th>
      <th>Saturday</th>
      <th>Sunday</th>
    </tr>
    <tr>
      <th>Morning</th>
      <td rowspan="2">Work</td>
      <td rowspan="3">Relax</td>
    </tr>
    <tr>
     <th>Afternoon</th>
    </tr>
    <tr>
      <th>Evening</th>
      <td>Dinner</td>
    </tr>
  </tbody>
</table>

Στο παραπάνω παράδειγμα, όλα τα δεδομένα πίνακα περιέχονται σε ένα στοιχείο σώματος πίνακα. 
Σημειώστε, ωστόσο, ότι οι επικεφαλίδες διατηρήθηκαν επίσης στο σώμα του πίνακα - θα το αλλάξουμε στo επομενο παράδειγμα.

### Κεφαλή πίνακα

Στο προηγούμενο παράδειγμα , οι επικεφαλίδες του πίνκα κρατήθηκαν μέσα στο σώμα του πίνακα. 
Ωστόσο, είναι είναι επίσης βολικό να χωρίζουμε τις επικεφαλίδες των στηλών του πίνακα χρησιμοποιώντας το element `<thead>`.

```html {2,8}
<table>
  <thead>
    <tr>
      <th></th>
      <th scope="col">Saturday</th>
      <th scope="col">Sunday</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">Morning</th>
      <td rowspan="2">Work</td>
      <td rowspan="3">Relax</td>
    </tr>
    <tr>
     <th scope="row">Afternoon</th>
    </tr>
    <tr>
      <th scope="row">Evening</th>
      <td>Dinner</td>
    </tr>
  </tbody>
</table>

Στο παραπάνω παράδειγμα, το μόνο νέο element είναι `<thead>`. 
Οι επικεφαλίδες του πίνακα περιέχονται στο εσωτερικό του. 

Επιπλέον, μόνο οι επικεφαλίδες στηλών βρίσκονται κάτω από το στοιχείο `<thead>`. 
Μπορούμε να χρησιμοποιήσουμε το *attribute* `scope` στα στοιχεία `<th>` για να δείξουμε αν ένα elememt `<th>` χρησιμοποιείται ως επικεφαλίδα `row` 
ή επικεφαλίδα `col`.

### Υποσέλιδο πίνακα

Το κάτω μέρος ενός πίνακα μπορεί επίσης να διαχωριστεί χρησιμοποιώντας το στοιχείο `<tfoot>`.

```html {21,27}
<table>
  <thead>
    <tr>
      <th>Quarter</th>
      <th>Revenue</th>
      <th>Costs</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th>Q1</th>
      <td>$10M</td>
      <td>$7.5M</td>
    </tr>
    <tr>
      <th>Q2</th>
      <td>$12M</td>
      <td>$5M</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <th>Total</th>
      <td>$22M</td>
      <td>$12.5M</td>
    </tr>
  </tfoot>
</table>
```

Στο παραπάνω παράδειγμα, το υποσέλιδο περιέχει τα σύνολα των δεδομένων στον πίνακα. 
Τα υποσέλιδα χρησιμοποιούνται συχνά για να περιέχουν αθροίσματα, διαφορές και άλλα αποτελέσματα δεδομένων.

### Συνοπτικά

Σε αυτό το μάθημα, μάθαμε πώς να δημιουργούμε έναν πίνακα, 
να προσθέτουμε δεδομένα σε αυτόν και να χωρίζουμε τον πίνακα σε μικρότερα μέρη που διευκολύνουν την ανάγνωση.

Ας δούμε τι έχουμε μάθει μέχρι τώρα:

- Το στοιχείο `<table>` δημιουργεί έναν πίνακα.
- Το στοιχείο `<tr>` προσθέτει σειρές σε έναν πίνακα.
- Για να προσθέσετε δεδομένα σε μια σειρά, μπορείτε να χρησιμοποιήσετε το στοιχείο `<td>`.
- Οι επικεφαλίδες του πίνακα αποσαφηνίζουν τη σημασία των δεδομένων. 
- Οι επικεφαλίδες προστίθενται με το στοιχείο `<th>`.
- Τα δεδομένα πίνακα μπορούν να εκτείνονται σε στήλες χρησιμοποιώντας το χαρακτηριστικό `colspan`.
- Τα δεδομένα πίνακα μπορούν να εκτείνονται σε σειρές χρησιμοποιώντας το χαρακτηριστικό `rowspan`.
- Οι πίνακες μπορούν να χωριστούν σε τρεις κύριες ενότητες: μία **κεφαλή**, ένα **σώμα** και ένα **υποσέλιδο**.
- Η κεφαλή ενός πίνακα δημιουργείται με το στοιχείο `<thead>`.
- Το σώμα ενός πίνακα δημιουργείται με το στοιχείο `<tbody>`.
- Το υποσέλιδο ενός πίνακα δημιουργείται με το στοιχείο `<tfoot>`.
