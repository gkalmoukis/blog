### Τι είναι η JavaScript;

Η **JavaScript είναι μία αντικειμενοστραφής γλώσσα προγραμματισμού** με κύρια χρήση την εκτέλεση σεναρίων σε ιστοσελίδες, ωστόσο έχουν αναπτυχθεί περιβάλλοντα που εκτελούν προγράμματα JavaScript χωρίς την χρήση του broswer όπως το **node.js** και το **Appache CouchDB**. Η πρώτη έκδοση της JavaScript ανακοινώθηκε τον Δεκέμβριο του 1995 από την Netscape Communications Corporation και την Sun Microsystems, Inc, μία γλώσσα συγγραφής σεναρίων για την web applications συμπλήρωνε την γλώσσα προγραμματισμού Java και ήταν ενσωματωμένη στην δοκιμαστική έκδοση (beta) του Netscape Navigator 2.0. Σχεδιάστηκε για την δημιουργία διαδικτυακών εφαρμογών που συνδέουν μεταξύ 
τους τα αντικείμενα και τους πόρους, όπως συναρτήσεις και μεταβλητές , σε client broswers και διακομιστές servers. Όπως η Visual Basic ήταν σχεδιασμένη για την ανάπτυξη σεναρίων από χρήστες που δεν είχαν μεγάλη εμπειρία στον προγραμματισμό , ωστόσο ο σχεδιασμός της JavaScript αντιπροσώπευε την επόμενη γενιά λογισμικού ειδικά για διαδικτυακές εφαρμογές, οι βασικές αρχές της ήταν ότι , σχεδιάστηκε για την δημιουργία δίκτυό-κεντρικών εφαρμογών , ήταν συμπληρωματική και πλήρως ενσωματωμένη στην Java και την HTML καθώς 
και ότι ήταν ελεύθερη για χρήση σε όλα τα λειτουργικά συστήματα (cross-platform).
 
### Εκδόσεις γλώσσας 
Το 1996 η Netscape υπέβαλε την JavaScript στην **ECMA International**, έναν Ευρωπαϊκό οργανισμό τυποποίησης προτύπων, με αποτέλεσμα να προκύψει η πρώτη έκδοση του προτύπου **ECMAScript**, που παραμένει εώς σήμερα το πρότυπο της JavaScript. Το αρχικό πρότυπο έλαβε μια σημαντική ενημέρωση με την τρίτη σταθερή έκδοση του 
προτύπου το 1999. 

> Επί της ουσίας, η επίσημη ονομασία της **JavaScript** είναι **ECMAScript**, παρόλο που ο όρος JavaScript χρησιμοποιείται ευρέως.

Η πέμπτη έκδοση του προτύπου εκδόθηκε τον Δεκέμβριο του 2009 και η έκτη έκδοση που ενσωματώθηκαν πολλές κύριες προσθήκες , όπως μέθοδοι για την προσπέλαση JSON αρχείων, τον Ιούνιο του 2015. Η ΕCMAScript αποτελεί μία από τις πιο διαδεδομένες γλώσσες προγραμματισμού γενικό σκοπού, και η πιο γνωστή στις διαδικτυακές εφαρμογές καθώς είναι ενσωματωμένη στους περιηγητές διαδικτύου. 
 
### Πρότυπο ECMAScript - 262 

Η τελευταία έκδοση του προτύπου ECMAScript δημοσιεύτηκε το 2018, και έχει αρχίσει ανάπτυξη του επόμενης έκδοσης. Αποτελεί μία γλώσσα βασισμένη σε αντικείμενα, τα δομικά μέρη της γλώσσας και οι διαθέσιμες λειτουργίες της παρέχονται από δομές αντικειμένων. Ένα πρόγραμμα γραμμένο σε ECMAScript είναι ένα σύνολο από αντικείμενα που επικοινωνούν μεταξύ τους. Κάθε αντικείμενο είναι μία συλλογή από ιδιότητες με τα χαρακτηριστικά τους που καθορίζουν το τρόπο που κάθε ιδιότητα μπορεί να χρησιμοποιηθεί. Οι Ιδιότητες (properties) 
περιέχουν άλλα αντικείμενα, μεταβλητές ή συναρτήσεις. Οι ενσωματωμένοι (build-in) τύποι τιμών που μπορεί να πάρει μία μεταβλητή είναι `Undefined`, `Null`, `Boolean`, `Number`, `String` και `Symbol`, ενώ τα αντικείμενα είναι μέλη του ενσωματωμένου τύπου `Object`. Οι συναρτήσεις είναι αντικείμενα που μπορούν να κληθούν, οι συναρτήσεις που σχετίζονται με αντικείμενα μέσω μίας ιδιότητας ονομάζονται μέθοδοι. 

### Πρότυπο JavaScript Object Notation – JSON 

Το **JSON** είναι ένα πρότυπο ανταλλαγής δεδομένων μεταξύ λογισμικών, είναι εύκολη η γραφή και η ανάγνωση του από ανθρώπους και παράλληλα εύκολο να αναλυθεί (parse) και να παραχθεί (generate) απο ένα πρόγραμμα. Βασίστηκε σε ένα υποσύνολο της 3ή έκδοσης της **ECMAScript-262** το 1999. 

> Διάβασε για την [γραμματική JSON](/grammatiki-json).

Η ανταλλαγή δεδομένων με χρήση αρχείων JSON είναι διαδεδομένη πρακτική στην  προγραμματισμών λογισμικό γιατί χρησιμοποιεί γνωστές τεχνικές κοινότητα των σε γλώσσες 
προγραμματισμού όπως η C, C++, C#, Java, JavaScript, Perl και η Python. 
Υπάρχουν δύο πιθανές δομές ενός αρχείου JSON: 
1. Συλλογή από ζευγάρια **κλειδιών – τιμών**. Η Javascript αντιλαμβάνεται την συλλογή αυτή ως ένα object, ενώ άλλες γλώσσες ως λεξικό (dictionaty) ή πίνακα hash (hash table) . 
2. Λίστα τιμών. Η Javascript αντιλαμβάνεται μία τέτοια λίστα ως πίνακα, ενώ άλλες γλώσσες ως διάνυσμα, λίστα ή ακολουθία.
